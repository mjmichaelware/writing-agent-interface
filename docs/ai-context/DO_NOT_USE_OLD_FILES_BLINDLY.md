/sdcard/Download contains many outdated exports, audits, AI replies, PDFs, zips, and old code dumps.

Before using any old file:
1. Check its date.
2. Check whether the repo has newer code.
3. Check whether imports/exports still exist.
4. Check whether the file matches current directory structure.
5. Treat older generated code as suspect unless verified.
